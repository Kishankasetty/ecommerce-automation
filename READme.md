# 🛒 E-commerce Website Automation Framework

## 📌 Overview
Automation testing framework for an e-commerce website using Selenium WebDriver and TestNG.  
The project automates core functionalities like login, product selection, and add-to-cart workflow.

---

## 🚀 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

---

## ✅ Features
- Login Automation
- Add to Cart Functionality
- Page Object Model (POM)
- Reusable Framework Structure

---

## 📂 Project Structure
```text
src/test/java
├── base
├── pages
├── tests
└── utils
````

---

## ▶️ Run the Project

1. Clone the repository
2. Open project in Eclipse/IntelliJ
3. Update Maven dependencies
4. Run `testng.xml`
